[h2]Introduction[/h2]
[b]Consul[/b]: In ancient Rome, a consul was one of the two highest-ranking elected officials who jointly governed the republic. The term later came to denote high-ranking officials or diplomats in other contexts.

[b]Scriptum[/b]: This means "written" (from the verb scribere, "to write"). It can refer to a document, text, or written work.

Literally, [i]"Consul Scriptum"[/i] translates to "written consul" or "the consul's writing." In a creative or modern context, it might be used as a title for a fictional work, legal decree, or diplomatic text.

[h1]Scripting Console & Debugging Tool[/h1]

[b]No technical knowledge required![/b] While under the hood it is a full Lua scripting console, it comes with a collection of simple, ready-to-use "one-click" scripts. You can use it right away without ever writing a line of code.

[h3]Consul (Non-Technical)[/h3]
A window with a list of scripts that you can activate with a single click. This provides a non-technical way to use scripts included in the mod — without needing to interact with the console at all, making it more accessible and immersive.

[img]PLACEHOLDER_ATTILA/consul-consul.png[/img]

[h3]Included Scripts[/h3]
The following one-click scripts are bundled with the mod:
[list]
[*][b]Ad Rebellos[/b] — Spawns a rebellion in the selected settlement.
[*][b]Adice Provinciam[/b] — Transfers settlement ownership to another faction.
[*][b]Casus Belli[/b] — Forces a war declaration between two selected factions.
[*][b]Custodes Vocati[/b] — Exchanges the garrison of a settlement with a field army.
[*][b]Exterminare[/b] — Kills any selected character.
[*][b]Impetus[/b] — Fully restores action points for any selected character.
[*][b]Incrementum[/b] — Grants +1 Growth Point to the selected settlement.
[*][b]Pax Aeterna[/b] — Forces a peace agreement between two selected factions.
[*][b]Sedatio[/b] — Boosts public order by +10 across the selected settlement's province.
[*][b]Subiugatio[/b] — Forces vassalage between two selected factions.
[*][b]Vexatio[/b] — Penalizes public order by -10 across the selected settlement's province.
[/list]

[h3]Console[/h3]
The console gives you raw access to Lua and all the extra commands included with the mod. Type a slash command or a Lua snippet and click Send to execute it.

[img]PLACEHOLDER_ATTILA/consul-console.png[/img]

[h3]Scriptum (Technical)[/h3]
This allows you to write your own scripts, place them in the game folder, and execute them with a click. No need to restart the game — just save your file and click the button again to run the updated code immediately.

[img]PLACEHOLDER_ATTILA/consul-scriptum.png[/img]

[h3]Load order[/h3]
Modifies only the [b]all_scripted.lua[/b] entry point and should be placed above all other mods.
Load on top of everything.
[img]https://images.steamusercontent.com/ugc/16764158438205361713/E344F33DD0CCF0AE39FADEE8B8C3A88130F2E3E2/?imw=5000&imh=5000&ima=fit&impolicy=Letterbox&imcolor=%23000000&letterbox=false[/img]

[h3]Compatibility[/h3]
Compatible with virtually everything. ConsulScriptum hooks only into [b]all_scripted.lua[/b] and leaves all other files untouched — making it safe to run alongside any overhaul or content mod.
Save game compatible.
