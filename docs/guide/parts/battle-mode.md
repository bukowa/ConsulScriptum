### Battle Mode Support (Rome II & Attila)

The console does not show in battle by default because the battle interface is not loaded by default by the game engine.

**Workaround**: The mod can force the campaign UI (and thus the console) to load during campaign battles. This is achieved via a custom battlefield override command.

- **How to enable**: Run `/use_in_battle` while inside a campaign session, then start your battle. Note that this command **only works within an active campaign session**.
- **Status**: Experimental for both Rome II and Attila.
