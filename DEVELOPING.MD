# Developing

## Documentation Server

To run the docs locally with hot reload:

1. Open a terminal in `docs/`.
2. Install dependencies (first time only):

   ```bash
   npm install
   ```

3. Start the docs server:

   ```bash
   npm run docs:dev
   ```

4. Open the local site:

   - `http://localhost:5173/`

Notes:

- This project uses VitePress for documentation.
- Keep the terminal running while editing docs.
